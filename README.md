# Global Technician DaaS

A global Data-as-a-Service (DaaS) platform that provides structured, geo-mapped technician data via API.

## Purpose
This project builds a compliant, scalable API that allows platforms to discover skilled technicians worldwide without manual onboarding.

## Core Features
- Global technician discovery
- Geo-based search
- Masked contact access
- API-first architecture
- GDPR / privacy aware design
